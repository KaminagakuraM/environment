## Tmux Configuration

- install tmux

  - `brew install tmux`

- copy `tmux`'s repo folder into `~/.config/`

- source tmux config

  - `tmux source-file ~/.config/tmux/tmux.conf`

- make sure to install plugins from the `conf` file using `prefix + I`, while \
  inside a tmux session

  - prefix is set to `control + a`
